Inside the local terminal:
docker compose up -d --build

Go to for ros:
http://localhost:6080

Inside the browser terminal:
cd ~/catkin_ws
catkin_make
source devel/setup.bash

Open the project in unity
go to robotics
set: ROS IP -> your computers IP and ROS Port 10000
go to test scene

connecting to the robot:
connect via ethernet cable
you have to set your ethernet side as ip: 192.168.1.100
and subnet mask: 255.255.255.0
robot ip: 192.168.1.225
cd catkin_ws
source devel/setup.bash
terminal 1:
roslaunch xarm7_moveit_config realMove_exec.launch robot_ip:=192.168.1.225
terminal 2:
rosservice call /xarm/clear_err
rosservice call /xarm/set_state 0
rosservice call /xarm/set_mode 0
roslaunch ur10_mover xarm_server.launch tcp_ip:=0.0.0.0 tcp_port:=10000
